# Chatbot Project Guidance — LLM Gateway

GPT Project / Claude Project mode does not have direct access to the EKSAD LLM Gateway unless the platform itself is configured by the user.

Use this directory as governance reference only:

- Do not claim LiteLLM or model aliases are available inside chatbot-only mode.
- Use uploaded knowledge and the chatbot platform's selected model.
- Preserve alias language in docs so outputs can migrate to Hermes/future runtime later.
- Ask the user for evidence if live retrieval/gateway access is unavailable.
