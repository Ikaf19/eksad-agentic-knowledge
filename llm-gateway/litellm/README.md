# LiteLLM Reference Implementation

LiteLLM is the reference implementation for the EKSAD LLM Gateway contract.

This directory contains examples only. Do not copy these files into a live runtime without replacing placeholders through secure environment management.

| File | Purpose |
|---|---|
| `config.example.yaml` | Example virtual model aliases for LiteLLM. |
| `env.example` | Environment variable names only. |
| `docker-compose.example.yaml` | Optional reference topology for a gateway container. |

Runtime secrets belong in the runtime secret manager or local env file outside Git.
