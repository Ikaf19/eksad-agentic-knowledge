# Approval Gates

## Default gates

| Gate | Required before |
|---|---|
| BRD readiness | FSD/TSD elaboration |
| FSD approval | Implementation planning |
| TSD approval | Code implementation |
| QA evidence | Release sign-off |
| PM/release gate | Deployment |
| Security review | Exception/risk acceptance |

## Rule

Agent output can recommend, draft, or summarize. Approval must remain explicit and attributable.
