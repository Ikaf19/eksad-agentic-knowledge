# Portable Role Cards

Each file defines one EKSAD role in a runtime-neutral way.

Role cards define:

- Mission
- Ownership
- Non-ownership boundaries
- Required knowledge
- MCP capability boundaries
- Portability rules

Adapters must render or reference these role cards instead of inventing independent role definitions.
