# MCP Migration Contract

A new environment restores MCP intent as follows:

```text
git clone eksad-agentic-knowledge
→ read portable/mcp policy + matrix
→ run adapter-specific doctor
→ install approved binaries locally
→ create local env file from examples
→ render local runtime config
→ apply only after explicit approval
→ verify available tools
```

The Git repository is the desired-state source. Local runtime remains environment-specific.
