# Data Analysis Report

## Purpose

Decision-ready analytical narrative produced by the Data Analyst.

## Required sections

1. Document control
2. Business question and decision target
3. Source inventory and sensitivity
4. Metric definitions
5. Method / read-only evidence plan
6. Findings
7. Data quality issues
8. Interpretation and caveats
9. Recommendation / next action
10. Follow-up owners

## Readiness checklist

- [ ] Business question and decision target are explicit.
- [ ] Every metric has numerator, denominator, filters, and owner.
- [ ] Source freshness and quality caveats are documented.
- [ ] Claims separate fact, interpretation, and recommendation.
- [ ] No production mutation or unrestricted sensitive data extraction is required.
