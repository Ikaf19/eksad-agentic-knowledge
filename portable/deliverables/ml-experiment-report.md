# ML Experiment Report

## Purpose

Reproducible experiment/model evaluation report produced by Data Scientist.

## Required sections

1. Problem statement and task type
2. Business objective and model objective
3. Dataset and label definition
4. Baseline and experiment design
5. Metrics and acceptance threshold
6. Results and error analysis
7. Bias/privacy/security/drift risks
8. Reproducibility notes
9. Recommendation and deployment gate status

## Readiness checklist

- [ ] Baseline comparison is present.
- [ ] Split/evaluation strategy is stated.
- [ ] Data leakage risks are assessed.
- [ ] Deployment is not recommended without gate evidence.
